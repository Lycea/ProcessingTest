What is it ?

In one sentence using randomness as a friend,not enemy! Or:
Make randomness less chaotic with some rules ore rulesets.

"Procedural content generation (PCG) is the programmatic generation of game
content using a random or pseudo-random process that results in an
unpredictable range of possible game play spaces"
