Visual studio code Plugins

This is a small listing of the visual studio code plugins that I use for editing this page that are
kind of usefull

  • CSSTree validator: this corrects the css files and tells you that there is something wrong
  • IntelliSense for Css class names: More css intellisense
  • Live HTML Preview: have a preview in a tab ... works more or less
  • Live server: Small server to have a version preview, actualieses on save what is really neat
  • ps5js Snippets: snippets for java scrip processing, really helpfull
  • Processing Language: for general processing, can mostly also be used for js variation
  • Python: for the python files, selfexplaining I guess ?
  • VS Color picker: for picking css colors directly in a picker
