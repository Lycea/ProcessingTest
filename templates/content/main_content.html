Usefull VScode Plugins for web developement

some years ago

This is a small listing of the visual studio code plugins that I used for editing this page back in the time of VScode usage
Most of them I would recomend using even today !

  • Live server: Small server to have a version preview, actualieses once something has been adjusted in a page
  • CSSTree validator: this corrects the css files and tells you that there is something wrong
  • IntelliSense for Css class names: More css intellisense
  • ps5js Snippets: snippets for p5js verry usefull for codecompletion and the likes
  • Processing Language: for general processing, can mostly also be used for js variation to have nice snippets
  • VS Color picker: for picking css colors directly in a picker
